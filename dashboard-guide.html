한눈에 보는
프로젝트 대시보드

각자 주간 리포트만 쓰면, 연구책임자는 대시보드 한 곳에서 프로젝트별 전원 현황을 봅니다. 누가 무엇을 어디까지 했는지, 막힌 건 없는지 바로 확인합니다.

📊 대시보드 열기 → 작동 방식 보기
01 · 작동 방식

한 번 쓰면, 대시보드에 뜬다

각자 익숙한 협업 도구(주간 리포트 DB)에 한 번만 작성하면, 별도 입력 없이 대시보드에 자동 반영됩니다. 프로젝트 현황·제출 여부·막힌 것이 한 화면에 모입니다.

작성주간 리포트 DB → 실시간 연동~1분 → 대시보드 반영프로젝트별 집계 → 연구책임자일괄 모니터링
💡 데이터 연동은 Notion으로 예정되어 있습니다. 연결 전에는 대시보드가 구조 확인용 대기 상태로 표시됩니다.

03 · 프로젝트로 보면

흩어진 일을 한 맥락으로

개인의 주간 활동은 흩어지기 쉽습니다. 프로젝트 단위로 모아 보면 각 과제가 어디까지 왔는지, 누구의 손이 필요한지 또렷해집니다.

나에게

내 일이 맥락 속에

주간 활동이 프로젝트 진행으로 연결돼 보이고, 내 기여가 또렷이 드러납니다.

동료끼리

서로 진행이 보인다

같은 프로젝트 멤버의 진행이 보여 자연스러운 협업과 자극이 됩니다.

연구책임자

제때 돕는다

각 과제 맥락을 알고 있어, 막힌 곳에 정확하고 빠르게 지원할 수 있습니다.
